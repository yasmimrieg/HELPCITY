# HELPCITY

## Problema
Dificuldade da população em informar problemas urbanos.

## Solução
Sistema web para cadastro e acompanhamento de ocorrências.

## Tecnologias
- HTML
- CSS
- JavaScript
- JSON Server

## Funcionalidades
- Cadastro de ocorrências
- Listagem de ocorrências
- Edição de ocorrências
- Exclusão de ocorrências

## Banco de Dados
JSON Server (db.json)

## Monetização
Assinatura mensal para prefeituras.

## Como Executar

1. Abrir a pasta back-end
2. Executar:

npx json-server --watch db.json --port 3001

3. Abrir index.html
